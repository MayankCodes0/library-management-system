# 📚 Library Management System

A simple **Library Management System** built using **Core Java** that demonstrates the practical implementation of **Object-Oriented Programming (OOP)**, the **Java Collections Framework**, and **Swing (`JOptionPane`)**. The application allows users to manage books by adding, issuing, returning, and viewing library records through an interactive interface.

---

## ✨ Features

* Add new books to the library.
* View all books with their details.
* Issue books to users.
* Return previously issued books.
* Track book availability.
* Store borrower information.
* Record book issuance timestamps using `LocalDateTime`.
* Interactive user interface using `JOptionPane`.

---

## 🛠️ Tech Stack

* **Language:** Java
* **GUI:** Swing (`JOptionPane`)
* **Core Concepts:** Object-Oriented Programming (OOP)
* **Data Structure:** `ArrayList` (Java Collections Framework)
* **Date & Time:** `LocalDateTime`

## 🚀 Getting Started

### Prerequisites

* Java JDK 8 or later
* Any Java IDE (IntelliJ IDEA, Eclipse, NetBeans, VS Code) or the command line

### Run the Project

1. Compile the project.

```bash
javac *.java
```

2. Run the application.

```bash
java LibraryManagementSystem
```

## 📖 Functionalities

* **Add Book**

  * Add a new book with its title and author.

* **Issue Book**

  * Issue an available book to a user.
  * Record the borrower's name.
  * Store the issue date and time.

* **Return Book**

  * Return an issued book.
  * Update its availability status.

* **View Books**

  * Display all books with:

    * Book ID
    * Title
    * Author
    * Availability Status
    * Borrower Details (if issued)
    * Issue Timestamp
