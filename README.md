# 📚 Library Management System (Java)

A simple console-based Library Management System built using Java to manage book records and basic library operations such as adding, issuing, and returning books.

---

## 🚀 Features

* Add new books with title and author details
* Issue books to users with issuer tracking
* Return issued books and update availability
* View all books with their current status
* Track issuing date and time using built-in date utilities

---

## 🛠️ Tech Stack

* Java (Core Java)
* Object-Oriented Programming (OOP)
* Java Collections Framework (ArrayList)
* LocalDateTime API

---

## 🧠 Concepts Used

* Classes and Objects
* Constructors and Method Overloading
* Encapsulation
* Interface with Default Methods
* List Data Structure for dynamic storage

---

## 📂 Project Structure

* `Book` class → Handles book details, status, and date tracking
* `UtilitiesMethod` interface → Provides method to display books
* `LibraryManagementSystem` class → Main driver class with menu-based operations

---

## ▶️ How to Run

1. Clone the repository
2. Compile the Java file:

   ```
   javac LibraryManagementSystem.java
   ```
3. Run the program:

   ```
   java LibraryManagementSystem
   ```

---

## 📌 Sample Operations

* Add a new book
* Issue a book to a user
* Return a book
* Display all books with status

---

## ⚠️ Limitations

* Console-based (no GUI)
* Data is not persistent (resets after program ends)
* Basic validation for user inputs
